常见问题

睿尔曼提供哪些产品与解决方案？ south_east

睿尔曼可提供关节模组、机械臂、机器人整机、末端执行器、生态配件等产品，并面向自动化与具身智能场景提供相关系统平台级的解决方案支持。现有产品体系覆盖 RM、RML、ECO、RX 等系列机械臂，复合升降机器人、RealBOT 轮式人形机器人，以及配套关节产品与开发支持能力

睿尔曼支持哪些开发环境与通信协议？ south_east

睿尔曼支持多种通信协议，包括 JSON、Modbus 等，并提供 C、C++、C#、Python 等多语言 SDK 支持。同时兼容 ROS 与 ROS2，便于客户将睿尔曼产品集成至不同的软件系统与机器人开发流程中

是否提供 SDK、示例 Demo 和可下载的开发资料？ south_east

是的。睿尔曼提供快速上手资料、API 文档、ROS / ROS2 功能包、示例 Demo，以及二次开发资料包、产品目录、3D 模型、尺寸图等可下载资源，帮助客户更高效地完成产品评估、系统集成与应用开发

是否支持二次开发与定制化集成？ south_east

是的。睿尔曼提供开放的 API 接口、多语言 SDK 及较为完善的技术文档体系，可支持客户开展二次开发与定制化集成。同时，相关资源覆盖机械臂、关节模组、AI 能力及数据训练等方向，为客户构建自身应用与部署方案提供支撑
